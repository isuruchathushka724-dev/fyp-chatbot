# 🚀 Streamlit Development Demo App
### IT41043 | Horizon Campus 2026

A comprehensive Streamlit learning application covering 
10 sessions from basics to AI/ML deployment.

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **AI Model:** Llama 3
- **Deployment:** Streamlit Cloud
- **Language:** Python

## 📚 What's Covered
| Session | Topic |
|---------|-------|
| 1 | Introduction & Setup |
| 2 | Text & Display Elements |
| 3 | Input Widgets |
| 4 | Layouts & Containers |
| 5 | Data & Visualisation |
| 6 | Session State & Forms |
| 7 | Multi-Page Apps |
| 8 | Caching & Performance |
| 9 | AI/ML Integration (Llama 3) |
| 10 | Deployment |

## 🚀 Run Locally
```bash
git clone https://github.com/isuruchathushka724-dev/fyp-chatbot
cd fyp-chatbot
pip install -r requirements.txt
streamlit run app.py
```
